# Phase 8 Final Remaining Scientific Limitations

1. The concat and repair studies use controlled synthetic VAR settings. They
   establish a measurement failure and a bounded trade-off, not universal
   performance on arbitrary real systems.
2. The coverage audit diagnoses whether a declared graph score is supported by
   the model's routes, coordinates, and horizon. It does not prove causal
   identifiability, causal sufficiency, or invariance to hidden confounding.
3. Corrected mask/shuffle tests measure sensitivity to frozen perturbations. They
   do not define an invariant causal-contribution ratio.
4. The final lambda study used three data-seed analysis units for a pre-specified
   go/no-go decision. No lambda passed, so no held-out confirmation or positive
   method-effectiveness claim is available.
5. Legacy filtered-coordinate ISTF-Mamba evidence is diagnostic only and cannot
   support graph-recovery, benchmark, Mamba, or ISTF-performance claims.
6. Author names, affiliations, funding, conflicts, contribution roles, and a
   public archival code/data URL remain submission metadata to be supplied by the
   authors.
